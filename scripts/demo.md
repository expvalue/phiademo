# phiademo demo flow

1. **Open the app** at `http://localhost:3000`.
2. **Landing feed**: highlight the “Recommended for you based on your friends” section.
3. **Click “Why this?”** on a card to show the breakdown (similarity, strength, recency, intent).
4. **Search**: try queries like:
   - `cozy travel bag`
   - `minimal desk setup`
   - `glow serum`
   - `trail running shoes`
5. **Filter by category** using the chips to show diversity controls.
6. **Explainability**: point out matched keywords and “Because X bought Y” reasoning.
7. **Wrap up** with the pgvector + OpenAI embeddings callout in the hero panel.
